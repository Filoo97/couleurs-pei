description: Publier sur Facebook et Instagram via l'Expert CM
---

Pour utiliser votre Expert CM, suivez ces étapes ou demandez-moi simplement de le faire :

1. Préparez votre texte et l'URL de l'image (doit être une URL publique).
2. Choisissez le mode : `feed` (par défaut) ou `story`.
3. Lancez la publication avec la commande suivante :

// turbo
```powershell
python expert_cm_couleurs_pei.py "Votre texte ici" "URL_Image_ici" "mode"
```

Les configurations (tokens, IDs) sont automatiquement chargées depuis le fichier `.env`.
